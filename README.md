# Tibetan Font Image Generator

A responsive online page for generating Tibetan font preview images as PNG.

## Local Preview

```bash
npm start
```

Open `http://127.0.0.1:5941/`.

## Deploy

This is a static site. On Netlify, use:

- Build command: empty
- Publish directory: `.`

The included `netlify.toml` already sets the publish directory.
